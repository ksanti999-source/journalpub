# Deploy Package

โฟลเดอร์นี้พร้อมสำหรับเผยแพร่หน้าเว็บแบบ static

ไฟล์ที่ต้องอัปโหลด:
- `index.html`
- `app.js`
- `styles.css`
- `data.js`
- `.nojekyll`

วิธีใช้งาน:
- GitHub Pages: อัปโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ไปยัง repository แล้วเปิดใช้งาน Pages
- Netlify/Vercel: อัปโหลดทั้งโฟลเดอร์นี้เป็น static site
- Web hosting ทั่วไป: คัดลอกไฟล์ทั้งหมดในโฟลเดอร์นี้ไปไว้ใน public web directory

หมายเหตุ:
- หากมีการแก้ข้อมูลหรือหน้าเว็บในโฟลเดอร์หลัก ให้ copy ไฟล์เวอร์ชันล่าสุดมาอัปเดตในโฟลเดอร์ `deploy` ก่อนเผยแพร่
